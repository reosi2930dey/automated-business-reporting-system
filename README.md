# Automated Business Reporting System

This project is about building a simple automated reporting system using SQL, Python, and Power BI.

## About the Project
I worked on this project to understand how raw business data can be converted into useful insights. Instead of doing everything manually, I automated the process of analyzing data and generating reports.

## Tools I Used
- SQL for analyzing the data  
- Python (Pandas) for processing and automation  
- Power BI for creating dashboards  
- Excel/CSV files for the dataset  

## What I Did
- Cleaned and organized the dataset  
- Wrote SQL queries to understand the data  
- Used Python to automate data processing  
- Created a Power BI dashboard to visualize the results  
- Focused on basic KPIs like trends and performance  

## Project Files
- CSV files → dataset used for analysis  
- queries.sql → SQL queries  
- report.py → Python script  
- dashboard.pbix → Power BI dashboard  

## Result
This project helped me understand how to automate reporting and use data for better decision-making.
